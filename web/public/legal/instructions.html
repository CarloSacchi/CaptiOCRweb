CaptiOCR User Guide

Here's a complete guide on how to use CaptiOCR, a tool that captures and reads text from screen areas, particularly useful for captions in Teams live chats or similar programs.

Prerequisites

Before getting started, make sure you have installed:

  • Python (if you're not using the executable version)
  • Tesseract OCR: the default path is C:\Program Files\Tesseract-OCR\tesseract.exe
  • Tesseract language data in the folder C:\Program Files\Tesseract-OCR\tessdata

Installation

  1. Extract or copy the CaptiOCR files to a folder of your choice
  2. If necessary, verify that the Tesseract paths in the file are correct for your system

Starting the Program

  • If you have the executable version (.exe), double-click on it
  • If you have the Python version (.py), run the script with Python

Main Interface

The main CaptiOCR window contains:

  • A language selector (English or Italian)
  • Debug options for logging
  • "Live Caption Optimization" option to improve caption recognition
  • Buttons to save and load settings
  • Green "Start" button to begin capturing

How to Use CaptiOCR

  1. Language Selection:
    • Choose the language of the captions you want to capture from the dropdown menu
  2. Advanced Options (optional):
    • Enable "Debug Logging" if you want to record detailed information
    • Enable "Live Caption Optimization" to improve live caption recognition
  3. Start Capturing:
    • Click on the green "Start" button
    • A transparent full-screen overlay will open
    • Click and drag to select the area containing the captions
    • Press Enter to confirm the selection (or ESC to cancel)
  4. During Capture:
    • A transparent yellow box will appear over the selected area
    • The program will start capturing text every few seconds
    • You can move the box by dragging it if needed
    • The latest text capture will be shown in the blue bar at the top
  5. Stopping Capture:
    • Click on the red "STOP" button in the capture box, or
    • Press Ctrl+Q from the keyboard
  6. Saving Results:
    • After stopping the capture, you'll be asked to enter a name for the saved file
    • Enter the desired name and click OK (or cancel to use the default name)
    • The processed file will be saved in the "captures" folder

Settings Management

  • Save Settings:
    • Configure your desired options
    • Click on "Save Settings"
    • Enter a name for the settings profile
    • The settings will be saved for future use
  • Load Settings:
    • Click on "Load Settings"
    • Select a previously saved profile
    • The settings will be automatically applied

Output Files

  • Original capture files are saved in the "captures" folder with the format capture_YYYY-MM-DD-HH-MM-SS.txt
  • Processed files have the format [custom_name]_capture_YYYY-MM-DD-HH-MM-SS_processed.txt
  • Debug logs (if enabled) are saved as ocr_debug.log

Practical Tips

  • For optimal caption recognition, ensure high contrast between text and background
  • Select an area slightly larger than the captions to ensure all text is captured
  • If the text is moving, you can drag the capture window to follow it
  • Use the "Live Caption Optimization" option when capturing real-time captions
  • If you notice slowdowns, try selecting a smaller area

Troubleshooting

  • If Tesseract doesn't work, verify that the paths are correct
  • If area selection doesn't work, try restarting the program
  • Detailed logs are in the "logs" folder for advanced troubleshooting

This program is particularly useful for recording captions or text that appears temporarily on the screen, allowing you to save and review information that would otherwise be lost.
