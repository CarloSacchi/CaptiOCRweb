CaptiOCR User Guide

Effective Date: February 2025

CaptiOCR captures and processes on-screen captions in real-time. It's ideal for meetings, presentations, and live sessions.

Supported Platform

Currently, CaptiOCR supports the latest Microsoft Windows operating systems only.

Prerequisites

Ensure you have the following installed:

  • Python (unless using the executable)
  • Tesseract OCR at C:\Program Files\Tesseract-OCR\tesseract.exe
  • Tesseract language data at C:\Program Files\Tesseract-OCR\tessdata

Installation

  1. Download and install the CaptiOCR executable or extract the Python script.
  2. We recommend installing it in your user documents folder due to current administrator privilege requirements. (Application signing pending)
  3. Verify Tesseract paths are correct in your configuration.

Starting CaptiOCR

  • Executable (.exe): Double-click to start.
  • Python script (.py): Execute via Python.

Main Interface

  • Language selector (English, Italian, French, Spanish, German, Portuguese)
  • Debug Logging
  • "Live Caption Optimization" for enhanced OCR of live captions
  • Settings management buttons (Save/Load)
  • Green "Start" button for capturing

How to Use CaptiOCR

  1. Language Selection:
    • Select the desired language from the dropdown menu. CaptiOCR checks if language files are available and prompts to download them if necessary.
  2. Advanced Options:
    • Enable Debug Logging for detailed logging.
    • Enable "Live Caption Optimization" for improved recognition accuracy.
  3. Capture Process:
    • Click "Start (Select Area)" and draw a rectangle over the captions area.
    • Confirm selection with Enter (cancel with ESC).
    • A transparent yellow window will indicate the active area.
    • OCR results will be displayed in real-time.
  4. Stopping Capture:
    • Press Ctrl+Q or click the red "STOP" button.
  5. File Saving:
    • Upon stopping, provide a custom filename or use the default.
    • Files are saved in the "captures" directory.

Settings Management

  • Use "Save Settings" and "Load Settings" under the File menu for managing preferences.

Output Files

  • Original captures: capture_YYYY-MM-DD-HH-MM-SS.txt
  • Processed files: [custom_name]_capture_YYYY-MM-DD-HH-MM-SS_processed.txt
  • Debug logs: ocr_debug.log

Troubleshooting

  • If OCR issues occur, confirm Tesseract installation paths.
  • Restart application if issues persist.
  • Detailed logs are located in the "logs" folder.

CaptiOCR is ideal for recording temporary on-screen text, particularly for meetings or presentations.
