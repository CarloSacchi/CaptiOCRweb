Privacy Policy

Effective Date: february 2025

CaptiOCR is an offline, local-only application. It does not collect, store, or transmit any personal data. All OCR operations run locally, ensuring full user privacy.

How CaptiOCR Works

  • No Cloud Processing: All OCR runs on your device, with no external data transmission.
  • No Tracking: No user activity, analytics, or telemetry data is collected.
  • Local File Storage: Extracted text is saved only on your local machine.

Security

CaptiOCR is open-source and must be downloaded from official sources to avoid modified versions that could introduce vulnerabilities.
